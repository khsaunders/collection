Pearl

Discover and support businesses in your neighborhood view the live site here **not optimized for desktop use**

Build overview

  • JavaScript
  • jQuery
  • AngularJS
  • SASS
  • HTML5

The goal

Make an application that allows users to search for and get results about locally owned businesses unique to their city. Features include user profile creation with the help of OAuth, Google Maps integration, and customization.

The context

I worked with two other front-end developers and two back-end developers in this exciting, collaborative experience. We only had three weeks to complete this project, so we had to make every second count.

The results

Over the course of those three packed weeks, I interacted with the API my back-end teammembers built, dynamically added content to our one page app with the magic of AngularJS, and learned a whole lot about the Agile development process.
